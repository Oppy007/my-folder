# AWS Fundamentals Lab

## Core AWS Services Overview

### EC2 (Elastic Compute Cloud)
Virtual servers in the cloud. Key concepts:
- Instances: Virtual machines with configurable CPU, memory, storage
- AMIs: Pre-configured templates for launching instances
- Security Groups: Virtual firewalls controlling traffic

### S3 (Simple Storage Service)
Object storage service. Key concepts:
- Buckets: Containers for storing objects
- Objects: Files and their metadata
- Durability: 99.999999999% (11 nines)

### IAM (Identity and Access Management)
Security service for access control. Key concepts:
- Users: Individual identities
- Policies: JSON documents defining permissions
- Roles: Temporary credentials for services/applications
- Principle of Least Privilege: Grant only necessary permissions
